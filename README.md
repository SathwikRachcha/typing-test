TypeSpeed Pro ⌨️
TypeSpeed Pro is a high-performance, minimalist typing assessment tool designed for developers and touch-typing enthusiasts. It provides real-time feedback on typing speed and accuracy while challenging users with a unique "Blind Mode" to master muscle memory.

🚀 Key Features
Real-Time Analytics: Calculates Words Per Minute (WPM) and accuracy dynamically as you type.

Blind Typing Mode: A specialized training mode that hides typed characters and the cursor, forcing users to rely entirely on touch-typing skills.

Persistent High Scores: Integrated with the Web Storage API (localStorage) to track your all-time best metrics, including High WPM, Best Accuracy, Max Correct characters, and Minimum Mistakes, across browser sessions.

Customizable Sessions: Offers 30s, 60s, and 120s testing intervals to suit different practice needs.

Developer-Focused UI: Built with a sleek dark theme and utilizes the JetBrains Mono typeface for maximum readability and a professional terminal aesthetic.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Properties/Variables).

Logic: Vanilla JavaScript (ES6+).

Storage: Browser localStorage for persistent data.

Typography: JetBrains Mono via Google Fonts.

📖 How to Use
Select Your Time: Choose between 30s, 60s, or 120s to start a session.

Toggle Blind Mode (Optional): Enable the "Blind Typing Mode" if you want to practice without visual feedback.

Start Typing: Click the input field and begin typing the displayed text.

Review Results: Once the timer hits zero, your session stats will be published in the bottom dashboard, and your high scores will update automatically if you set a new record.
